# PWA Icons

Place PNG icons of the following sizes here:
  icon-72.png   (72×72)
  icon-96.png   (96×96)
  icon-128.png  (128×128)
  icon-144.png  (144×144)
  icon-152.png  (152×152)
  icon-192.png  (192×192)
  icon-384.png  (384×384)
  icon-512.png  (512×512)

You can generate all sizes from a single source image using:
  npx pwa-asset-generator logo.png ./icons

Or use any online PWA icon generator with your hospital logo.
